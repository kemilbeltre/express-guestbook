# Express Guestbook

This repo contains a simple example that you can use for testing.

## Getting started

### Prerequistes

1. git
1. Node: any 12.x version
1: Yarn

### Instalation

1. `npm install`
1. `open http://localhost:3000` to open the site in your favorite browser
